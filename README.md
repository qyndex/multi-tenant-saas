# Multi-tenant SaaS

Full-stack multi-tenant SaaS with org management, team roles, and Stripe billing.

**Stack:** Nextjs | **Complexity:** enterprise
**Tags:** `saas`, `multi-tenant`, `prisma`, `billing`, `teams`, `nextjs`

## Quick Start

```bash
npm install
npm run dev
```

## Docker

```bash
docker compose up --build
```
